# 🚀 拓麻歌子 (Tamagotchi) 虚拟宠物网页应用

一个基于Y2K美学风格的复古虚拟宠物游戏，完美复刻90年代电子宠物体验，融合现代Web技术实现沉浸式交互。

### 角色设定
- **资深前端工程师**：精通 React、TypeScript、Tailwind CSS、SVG 像素艺术设计
- **设计理念**：复古与创新的完美融合，Y2K美学与现代技术的结合

#### 外壳设计
- **形状**：卵形/椭圆形游戏机外壳
- **颜色**：粉紫色渐变 (#E0BBE4 → #D4C1EC → #F3D1F4)
- **效果**：立体阴影、高光反射、3D质感

#### 按钮布局
- **A键** (粉色)：选择/左移 - `#F472B6`
- **B键** (黄色)：确认/开始 - `#FBBF24`
- **C键** (粉色)：取消/右移 - `#F472B6`

#### 换肤功能
- **Change Skin**：上传本地图片作为游戏机面板背景
- **Default Skin**：恢复浅蓝色默认皮肤 `#B8E1FF`

#### 动态场景切换
- **客厅** (Living Room)：默认待机、睡觉、聊天、吃药
- **厨房** (Kitchen)：喂食菜单或正在吃东西
- **浴室** (Bathroom)：洗澡、查看属性状态

## 🐱 宠物系统

### 属性与代谢
- **饥饿值** (Hunger)：0-100，随时间下降
- **心情值** (Happiness)：0-100，受互动影响
- **健康值** (Health)：0-100，生病时下降
- **年龄**：从蛋到成年小猫的成长过程
- **便便系统**：超过3堆会生病

### 进化阶段
1. **蛋** (Egg)：初始状态，等待孵化
2. **幼体** (Baby)：刚孵化的小猫
3. **成年小猫**：完全成长的宠物
4. **幽灵** (Ghost)：健康值归零后的状态

## ⏱️ 番茄钟联动系统

### 深度集成逻辑
- **计时器**：15分钟专注计时
- **专注模式**：番茄钟运行时，宠物代谢完全暂停
- **学习奖励**：计时结束后，大幅增加宠物心情值
- **视觉反馈**：计时器位于游戏机顶部

## 🎮 核心功能菜单

### 7大主菜单项
1. **🍗 喂食** (Feed)：选择鱼、牛奶、肉等食物
2. **💡 灯光** (Light)：切换睡眠模式，代谢变慢
3. **🎾 游戏** (Play)：进入游戏选择菜单
4. **💊 医治** (Medicine)：治愈生病状态
5. **🚿 洗澡** (Bath)：清理屏幕上的便便
6. **📊 状态** (Stats)：进度条显示各项数值
7. **💬 对话** (Chat)：基于宠物状态生成可爱的小猫想法

### 子菜单系统
- **食物菜单**：🐟 鱼 (+30)、🥛 牛奶 (+15)、🍗 肉 (+40)、🍪 饼干 (+10)
- **游戏菜单**：贪吃蛇、火箭躲避游戏

## 🚀 迷你游戏 - 火箭躲避 (Rocket Dodge)

### 游戏元素
- **玩家**：火箭 🚀
- **障碍物**：陨石 🪨
- **赛道**：3条垂直航道（左、中、右）

### 核心逻辑
1. **生成规则**：陨石从上方随机生成并向下滚动
2. **公平性**：绝对不允许在同一横向行同时出现3个陨石（必须留出至少一条生路）
3. **难度梯度**：随着得分增加，陨石下落速度和生成频率逐渐加快
4. **控制方式**：A键左移，C键右移

### 技术实现
- **碰撞检测**：火箭与陨石在同一航道相遇即游戏结束
- **得分系统**：成功躲避的陨石计入分数
- **动态难度**：每5分提升一次难度

## 🛠️ 技术架构

### 前端技术栈
- **React 19** + **TypeScript**：现代前端开发
- **Vite**：快速的构建工具
- **Tailwind CSS**：实用优先的CSS框架
- **SVG 像素艺术**：可缩放的矢量图形

### 核心模块结构
```
src/
├── App.tsx              # 主应用组件
├── components/          # 可复用组件
│   └── PetSprite.tsx   # SVG像素宠物精灵
├── hooks/              # 自定义Hook
│   └── useTamagotchi.ts # 宠物状态管理
├── services/           # 服务层
│   └── chatService.ts # 对话服务（基于状态的固定对话）
├── types.ts            # TypeScript类型定义
└── public/             # 静态资源
    ├── scene_living.png    # 客厅背景
    ├── scene_kitchen.png   # 厨房背景
    └── scene_bathroom.png  # 浴室背景
```

### 状态管理系统
- **宠物状态层**：生命周期、属性、行为状态管理
- **游戏状态层**：小游戏逻辑、得分、难度控制
- **UI状态层**：菜单选择、动画状态、对话消息

### 音效系统
- **Web Audio API**：实现选择、确认、取消音效
- **音效类型**：
  - 选择音：880Hz正弦波，0.1秒衰减
  - 确认音：1200→600Hz方波，0.15秒衰减
  - 取消音：300Hz三角波，0.15秒衰减

## 🚀 快速开始

### 环境要求
- Node.js 18+

### 安装步骤
```bash
# 1. 克隆项目
git clone <repository-url>
cd y2k-virtual-pet

# 2. 安装依赖
npm install

# 3. 启动开发服务器
npm run dev

# 4. 访问应用
# 打开浏览器访问：http://localhost:3000
```

### 构建生产版本
```bash
npm run build
npm run preview
```
